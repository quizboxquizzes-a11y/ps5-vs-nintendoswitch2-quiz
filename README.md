Console Matchmaker is a sleek, single-page web application designed to help gamers navigate the choice between two massive gaming ecosystems: the high-fidelity powerhouse PlayStation 5 and the versatile, hybrid Nintendo Switch 2.

Instead of relying on generic tallies, the application uses an adaptive scoring matrix that matches user answers against real-world gaming habits, budget preferences, lifestyle requirements, and hardware preferences to deliver a personalized, data-driven system recommendation.
